# Fraud Prediction Model

Machine learning project for detecting fraudulent e-commerce transactions using Python.

## Project Files

- `Fraudulent_E-Commerce_Project.ipynb`: Main notebook including data cleaning, EDA, feature engineering, model training, and evaluation.

## Dataset

The dataset is not uploaded directly to this repository because the file size is too large.

Dataset link: [Google Drive Dataset](https://drive.google.com/file/d/1bnpUcBuJFpAPX0uaMLKl_KWsSoo92wkY/view?usp=sharing)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Evaluation Metrics

- Precision
- Recall
- F1-score
- PR-AUC
- Confusion Matrix
